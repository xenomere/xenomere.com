# Xenomere OU — B2B Corporate Website

Source code for the public-facing B2B corporate website of **Xenomere OU**, an Estonian-registered chemical and biochemical sourcing company (registry code 17507482).

**This is not a retail store.** No product catalog. No prices. No shopping cart.

## Site Structure

| Page | URL | Description |
|------|-----|-------------|
| Home | `/` | Landing page — capabilities overview, trust bar, CTA |
| About | `/about` | Company story, mission, key facts, values |
| Capabilities | `/capabilities` | Sourcing categories, 6-step process, industries served |
| Compliance | `/compliance` | EU regulatory framework, GMP, AML, GDPR, export control |
| Contact | `/contact` | Partnership inquiry form |
| Privacy Policy | `/privacy` | GDPR-compliant data protection |
| Terms & Conditions | `/terms` | B2B only, SEPA transfers, no crypto |
| Cookie Policy | `/cookies` | Minimal cookies, no tracking |
| Legal Notice | `/legal` | EU E-Commerce Directive imprint |
| Shipping & Delivery | `/shipping` | B2B logistics, cold chain, documentation |
| Returns & Refunds | `/returns` | Quality guarantee, third-party testing |
| Disclaimer | `/disclaimer` | Research use only |

## Design

- Static HTML + CSS — no frameworks, no JavaScript, no dependencies
- Deep navy + white + teal accent
- Clean European biotech aesthetic
- Mobile responsive
- Designed for bank/payment-processor compliance review

## Deployment

Static files. Deploy to any web server or CDN.
